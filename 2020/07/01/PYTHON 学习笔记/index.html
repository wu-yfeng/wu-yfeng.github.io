函数

print （）输出

输出函数

input （）输入

输入函数 （这里的信息只是提示显示信息）

eval（）评估

评估函数：去掉最外围符号，执行符号内的语句

eg.

range（）循环计数

产生循环计数序列

常和for in 连用，来控制循环

  • range（N）产生0到N-1的序列
  • range（M，N）产生M到N-1的序列

pow（）

x的y次方

  • x
  • y

round（）四舍五入

对x进行四舍五入，d是小数截取位数

有时用以解决去除浮点型尾数不确定性的问题

  • x
  • d截取位数

abs（）绝对值

divmod（）商余

返回商和余数

库

turtle 绘图库

窗体图形绘制

以窗体中心为原点建立坐标系

画笔方向默认向右

setup（）绘制窗口

用来绘制窗口

  • 宽度
  • 高度
  • x坐标
  • y坐标

goto（）点到点

画笔起始点为，窗口中心（0,0）

fd（）前进

控制画笔向前（从中心向左）

bk（）后退

控制画笔向后（从中心向右）

circle（）转弯

以半径r绘制角度为angle的弧形，实现拐弯

  • r
  • angle

默认圆心在画笔左侧，半径为负数则圆心在画笔右侧

r为正数则左转，负数为右转

seth（）改变方向

改变行进方向（只改变方向但行进）

  • angle（绝对角度）

left（）right（）改变方向

用左或右改变方向

  • angle（绝对角度）

数据类型

整型

  • 不存在取值范围
  • 二进制 0b/0B表示
  • 八进制 0o/0O表示
  • 十六进制 0x/0X表示

浮点型

  • 存在限制，但常规计算可忽略
  • 浮点数之间的运算存在不确定位数，不是BUG

复数

.eg

  • z.real获得实部
  • z.imag虚部

列表

  • 列表a，使用append（）方法可以在列表末尾添加元素

  • 使用extend（）方法可以在末尾添加一组元素

  • 使用insert（）可以在指定位置添加元素

  • 使用remove（）可以删除指定元素

  • 使用pop（）可以删除指定位置元素

  • 使用clear（）清空列表

  • 可对列表指定位置元素重新赋值，来改变这个值

  • sort（）将列表进行排序

    sort中可加两个参数，第一个指定特定的排序方法，第二个是否反转

  • reverse（）将列表顺序反转

  • count（）统计元素出现的次数

  • index（）查看元素位置

  • copy（）复制列表

异常

try—except

捕捉程序正常运行时会出现的一些异常，比如除0

else—-finally

不论程序是否出现异常都该执行的部分

定义函数

函数定义格式： def空格<名字>（参）：

参数是通过值来进行传递的，即形参变化不行实参

如果变量是可变对象（如列表），返回到调用程序后，该对象会呈现被修改后的状态【实际并没变化，而是将改变的返回，而之前的会等垃圾回收进行清除】

文件

  • 编码：encode（）

  • 解码：decode（）

  • 打开文件函数：open（name,mode）name：磁盘文件名 mode：打开模式

字典

  • 字典是python特有的高级数据结构
  • 是针对非序列类型而产生的
  • 字典是一个键值对的集合
    • 以键为索引
    • 同一个键信息对应一个值
  • 通过字典实现映射【通过任意键值查找集合中值信息的过程叫映射】

字典格式：

字典名={键名：值，…}

增加：字典名[新增的键名]=值

访问：字典名[键名]——将返回所对应的值

删除：del字典名[键名]

遍历：for key in 字典名：

​ print（key + “：” + str（字典名[key]））

判断键是否存在

细节

  • if后 冒号 为固定语法，表示条件判断正确执行之后的语句
  • x/y 计算结果为浮点型，要为整型，需变为x//y
  • x**y为幂运算，当y为小数时，则为开跟号
  • 切片字符串<字符串名>[M:N]，省略M，为从头开始；省略N，为到结尾
  • <字符串名>[M:N:K]，K表示第K位取字符
  • [：：-1]表示将字符串逆序
